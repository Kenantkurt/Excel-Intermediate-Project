# Excel Project Overview

This Excel project demonstrates various data analysis and visualization techniques. Each sheet in the workbook focuses on a specific feature or function.

## Sheet 1 – Creating Table
We converted our data into a table to make analysis more efficient, easier, and faster.

## Sheet 2 – Flash Fill
We explored the Flash Fill feature using different methods.

## Sheet 3 – Sorting
We organized our data using Excel's **Data → Sort** feature in various ways.

## Sheet 4 – Sorting Function
Here, we used Excel’s **SORT** formula to arrange data according to different criteria.

## Sheet 5 – Sorting by
This sheet demonstrates how to use **SORTBY** to organize data effectively.

## Sheet 7 – Unique
We applied the **UNIQUE** function to make our data more suitable for analysis.

## Sheet 8 – Filtering
We showcased filtering, an essential feature for data analysis, allowing faster and more efficient insights.

## Sheet 9 – Slicers
We created **slicers** to filter data more quickly and effectively.

## Sheet 10 – Subtotal Function
We used the **SUBTOTAL** function to make our data easier to analyze.

## Sheet 11 – Quick Analysis Tool
We demonstrated how to use the **Quick Analysis Tool**, create charts, and customize them.

## Sheet 12 – Sparklines
We applied the **Sparklines** feature, creating small charts for each data point.

## Sheet 13 – Pivot Table
We showed how to create a **Pivot Table** to analyze data more easily.

## Sheet 14 – Data Validation
We applied **Data Validation** to ensure that input data follows specific rules.

## Sheet 15 – Conditional Formatting
**Conditional Formatting** helps highlight important information and makes data analysis faster and more effective.

## Sheet 16 – Linking Data
We demonstrated how to link data across sheets to improve data analysis and workflow efficiency.
